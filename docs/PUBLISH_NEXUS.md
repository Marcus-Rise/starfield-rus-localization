# Публикация на Nexus Mods

> **Экспериментальный статус (PS5)**: загрузка кириллических шрифтов
> на PS5 через Creations **не проверена**. Подробности и риски описаны
> в `ARCHITECTURE.md` §Предзагрузка шрифтов.

Самостоятельная загрузка мода на Nexus Mods (не требует Windows).

## Шаги

### 1. Скачайте артефакты

Скачайте zip из [GitHub Releases](../../releases/latest).

### 2. Создайте мод на Nexus

1. Перейдите на https://www.nexusmods.com/starfield/mods/
2. **Upload a new mod**
3. Заполните:
   - **Название**: Starfield — Russian Localization (PS5/PC)
   - **Категория**: Localisation
   - **Описание**: Полная русская локализация
4. Загрузите zip-файл
5. Опубликуйте

### Авторство перевода

Если используется чужой перевод:
- [ ] Получено разрешение автора на переупаковку
- [ ] При сборке указан `--credit "Имя автора"`
- [ ] В `dist/CREDITS.txt` корректно указан автор
- [ ] В описании мода на площадке указан автор перевода

### 3. Обновление

1. Скачайте новый zip из GitHub Releases
2. Nexus → Manage Files → Add New File → загрузите новый zip
